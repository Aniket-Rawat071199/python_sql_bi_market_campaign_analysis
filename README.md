# Marketing Campaign Analysis Dashboard

##  Project Overview

This project analyzes marketing campaign performance across multiple channels, campaign types, demographic segments, companies, and time periods.

The objective of this analysis was to identify profitability drivers, optimize marketing spend, detect inefficiencies, and provide data-driven strategic recommendations.

The dashboard was built using Power BI with DAX measures and business-focused KPIs.

---

##  Dataset Overview

The dataset includes:

- Campaign Details: campaign_id, company, campaign_type
- Targeting Information: gender, age_min, age_max
- Performance Metrics: conversion_rate, ROI
- Cost Metrics: acquisition_cost, CPC, CPM
- Time Features: year, month, quarter
- Channel & Location: channel_used, location

---

##  Business Questions Addressed

- Which channel generates the highest ROI?
- Which campaign type drives the best conversion rate?
- Which demographic segment is most profitable?
- Where is marketing budget being inefficiently spent?
- Which quarter performs best?
- Which company operates with the most cost-efficient structure?
- Are there high-cost but low-conversion campaigns?
